主题

主题是 Github 上的 Drake，默认就很好看，根据自己的需求做了一些修改。

自定义修改 CSS

在 Typora 的主题文件夹下新建一个 base.user.css 文件，Typora 每次启动都会加载这个文件。

修改内容如下：

  • H1 左对齐
  • 去除 H2 底部横线
  • 代码块增加圆角和描边
  • 文档宽度减小
  • 公式大小缩小
  • 段落改为两端对齐
  • 行内代码增加背景
  • 图片加上圆角
  • 滚动条圆角和颜色修改
